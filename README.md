# lehrer-tools Marketplace

Claude Code Plugin-Marketplace für Lehrkräfte.

## Installation

```
/plugin marketplace add lehrer-tools/marketplace
/plugin install lehrer-agent
```

## Plugins

| Plugin | Beschreibung |
|---|---|
| [lehrer-agent](https://github.com/lehrer-tools/lehrer-agent) | Unterricht planen, Tests erstellen, Material sammeln, Verwaltungsvorschriften nachschlagen |
